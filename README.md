# Get-Pgyer-iPA
with this bot you can get the .ipa file from any pgyer.com/xxxx app links

Usage: Use the /start command, send the bot a pgyer.com link (example: https://pgyer.com/adhoc)
the bot will extract the ipa link and return the information to you

No support for password protected apps yet
